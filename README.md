# pizza-sales-analysis
# Pizza Sales Analysis Using Python

This project analyzes pizza sales data using order-level logic to uncover
customer behavior, product performance, and time-based trends.

## Tools Used
- Python (pandas, matplotlib, seaborn)
- Excel (logic validation)
- Power BI Desktop
- Microsoft Fabric

## Key Insights
- Average of ~2.3 pizzas per order
- Strong seasonality across months
- Medium and Large sizes dominate orders

📌 Full analysis available in the Jupyter Notebook.
